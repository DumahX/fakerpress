Faker Press is a clean way to generate fake data to your WordPress instalation, great for developers who need testing

== Description ==

**Note: This plugin requires PHP 5.3 or higher to be activated.**

Whenever you create a new Theme or Plugin you will always need to create custom data to test whether your plugin is working or not, and as a Developer ourselves we had this problem so far.

Our goal with this plugin is to fill this gap where you have problem with a good solution both for Developers and for Users of WordPress.

**Create Fake Data for:**

 * Posts
 * Pages
 * Custom Post Types
 * Users
 * Themes
 * Plugins
 * Tags
 * Categories
 * Custom Taxonomies
 * Settings
 * Menus
 * Media Library
 * Widgets
 * Comments
